## ✨ Core Features

- 🥊 Kit System (Boxing, Crystal, Axe, and more)
- 🎛️ GUI-Based Kit Editor
- 🌐 Queue Ping Range System
- 📊 Animated Scoreboards
- 🧩 PlaceholderAPI Support
- 🎆 Custom Death Messages
- 🛠️ Management Menus
- 🏆 Instant Leaderboards Updates
- 🧩 Highly Customizable
- ⚡ Dynamic Arena Creation
